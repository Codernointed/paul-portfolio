# Mobile app screenshots

Drop PNG/JPG screenshots in THIS folder and they auto-appear in the
"Mobile Apps" desktop app — no code changes needed.

## Naming convention (important)

Name each file `<prefix>-<number>.png`, where `<prefix>` matches the app:

| App               | Prefix     | Example files                     |
| ----------------- | ---------- | --------------------------------- |
| Somayen Delivery  | `somayen`  | `somayen-1.png`, `somayen-2.png`  |
| Utim8             | `utim8`    | `utim8-1.png`, `utim8-2.png`      |

Files are shown in filename order, so number them in the order you want.
Portrait phone screenshots (e.g. 1080×1920) look best. 3–6 per app is plenty.
